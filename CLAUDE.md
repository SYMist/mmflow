# Money Flow Planner (mmflow)

## 제품 비전
가구 경제를 꾸리는 가계부 및 경제 생활 플래닝 도구.
수입-지출 흐름을 노드-엣지 그래프로 시각화하여 자금 흐름을 직관적으로 설계한다.

## 비즈니스 모델
- 구글 애드센스를 통한 광고 수익화
- 서버 기반 서비스 (회원/인증 필요)

## 기술 스택
- Frontend: React 19 + TypeScript + Vite 7 (SWC)
- 상태관리: Zustand
- 캔버스: @xyflow/react (React Flow)
- 저장: localStorage (`mfv3:` prefix, 캔버스별 x 월별, 향후 서버 이전)
- 폰트: Pretendard Variable

## 모델 v3 (2026-05-05 도입)

### 노드 3종
- **출발 노드 (source)**: 외부 수입 (월급 등)
- **도착 노드 (destination)**: 돈이 모이는 계좌. 정확히 1개 부모 (1:1 강제, 위반 시 자동 복제 + 그룹 제안)
- **서브 노드 (sub)**: 도착 노드의 자체 수익 (배당/이자), 1:1 종속, 우상단 고정 함께 이동

### 그룹
- 동일 (이름+계좌, 정규화 후) 도착 노드만 묶음
- 매칭 정규화: trim + 연속 공백 단일화 + 대소문자 무시
- 1 그룹 멤버십, 합산 자동 계산
- 드래그앤드롭으로 겹쳐 형성, 또는 자동 복제 모달에서 수락

### 영역 (멀티 멤버 캔버스)
- 좌측 멤버1 | 가운데 공유 | 우측 멤버2 (좌표 기반 자동 판정)
- 비공유 노드는 타 영역 노드와 연결 차단
- 영역 경계 드래그 → 공유 전환
- 비공유 그룹이 영역 가로지름 → 자동 공유

### 캔버스
- 메타데이터: 이름, 멤버 타입(가족/친구/연인/배우자), 멤버 이름, 목표
- 멤버 추가/삭제 불가 (생성 시 명시적 경고 + 확인)
- 월별 노드/엣지/그룹 분리 저장
